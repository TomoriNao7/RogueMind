# RogueMind — 明日方舟集成战略助手

> **当前为最初测试版（v0.1.0）**  
> 首发支持界园主题，后续版本将覆盖所有肉鸽主题（萨卡兹的无终奇语、探索者的银凇止境、水月与深蓝之树、傀影与猩红孤钻），并持续完善 Agent 策略分析能力。

一款明日方舟**集成战略（肉鸽）**桌面助手，帮助玩家做出更优决策。

![icon](assets/icon_256.png)

---

## 功能

- **策略推荐**：根据阵容和藏品，推荐招募干员、精二优先级、藏品选择、通宝配置
- **Agent 深度分析**：接入 LLM 进行多轮对话策略分析，支持 Claude / GPT / DeepSeek / Kimi / Qwen / Ollama
- **RAG 知识检索**：内置界园流程文档，语义 + 关键词混合检索
- **会话管理**：每局独立会话，多轮对话上下文，3 天自动清理

## 快速开始

### 无需安装任何依赖

后端已用 PyInstaller 打包为独立 exe（内含 Python + 所有依赖 + BGE 模型）。

### 启动应用

双击 `start.bat`，稍等约 30 秒（模型加载），RogueMind 窗口自动弹出。

### 配置 LLM（可选）

进入 **设置** → 选择模型提供商 → 填入 API Key → 保存。

### 日常使用

1. **状态录入** → 填写当前游戏状态
2. **策略推荐** → 获取招募/精二/藏品/通宝推荐
3. **Agent 分析** → 多轮对话深度策略分析

## 页面说明

| 页面 | 功能 |
|------|------|
| **状态录入** | 选择主题/分队/层数/难度，搜索添加已招募干员（含练度/伺烛客）、藏品、通宝，填入资源数量 |
| **策略推荐** | 四个 Tab：招募推荐、精二推荐、藏品推荐、通宝推荐。点击「获取推荐」即可 |
| **Agent 分析** | 多轮对话界面。先在上方录入状态，再在此页面输入问题，Agent 结合 RAG 知识库和 LLM 给出策略建议 |
| **设置** | 配置 LLM（选择模型 → 填 API Key），查看应用版本信息 |

## 技术栈

| 层级 | 技术 |
|------|------|
| 桌面壳 | Electron |
| 前端 | React + TypeScript + Vite |
| 后端 | FastAPI + Python 3.10 |
| 向量检索 | FAISS + BGE-small-zh-v1.5 |
| 关键词检索 | BM25 |
| 融合排序 | RRF (Reciprocal Rank Fusion) |
| 会话管理 | Redis（自动降级 JSON） |

# 界园肉鸽大致流程

## 本主题下的独特机制与机制调整

### 1.扩展难度

本次主题依旧继承了进阶式难度等级，每增加1个难度等级都会叠加难度附加条件，同时增加得分倍率，但有以下调整：

- 初始的0级难度得分改为无负面的±0%，去除辅助增强收藏品；
- 存在与挑战难度相关的“游览者增益”，通过强化收藏品、分队等助力探索。

※关于难度等级的详细信息参见[开始探索](https://prts.wiki/w/岁的界园志异#开始探索)一节。

### 2.岁兽残识

界园中进入误入奇境节点后可以选择获得基础3烛火进入岁兽残识或者选择1希望离开，选择第一条后可继续选择保留烛火至下次进入后离开，或直接进入，烛火为在岁兽残识地图中行动步数，归零后直接离开岁兽残识。岁兽残识内有各种独特的奖励及战斗节点，部分结局的前置道具在其中获得。而其中的战斗节点仅伺烛客可进入作战。

### 3.伺烛客、招募券留存

参照秉烛仪式燃烛，获取**伺烛客**临时许可的干员，即可定心凝神，如岁后亦可在残识之中出战。

成为**伺烛客**，方法众多，但大多为三法：

- **存券燃烛法**

​	除初始招募外，玩家获得招募券时，若无意即时招募，亦可在画面右上角“留存”于手中。
​	待行至【指点迷津】处，可出示留存券，行特殊招募。
​	此时不仅希望花费-2，被招募干员也可立即成为伺烛客。
​	※不限是否已招募/已进阶；即使干员已进阶，也可通过特殊招募成为伺烛客（此时不消耗希望）。
​	※需注意：留存券数初始上限4张（可因部分效果改变留存上限）。
​	※需注意：对于招募券包含的随机效果（如随机直升、随机获取干员、临时招募出现状态等），在出示时会重新进行	随机判定。
​	※需注意：留存达到上限时，无法继续留存；留存时，亦无法选择替换已留存的券。
​	※需留意：部分奇遇也可行特殊招募。

- **明烛试炼法**

​	若是不想花费招募券，在点亮【古今学识】的《伟业篇》后，玩家也可在【指点迷津】处选择一场试炼。
​	在此特殊战斗中，若明烛台上的干员存活，即可成为伺烛客。

- **擂台伺烛法**

​	若已点亮【古今学识】的《论道篇》解锁【狭路相逢】节点，可在此节点选择以战伺烛进入特殊的狭路相逢对局。
​	在该对局中胜利时，上擂的所有干员成为伺烛客。

除此之外也有其它方法，此处篇幅有限不再详述。

※关于伺烛客相关的详细信息，参见[节点：指点迷津](https://prts.wiki/w/岁的界园志异#事件节点)、[节点：误入奇境](https://prts.wiki/w/岁的界园志异#事件节点)。

### 3.通宝，钱盒

通宝有三种**币制**，代表其效果的大致划分：

- **花钱** 能带来稳定收益；

- **衡钱** 效果较为均衡；

- **厉钱** 较为激进，效果将难以捉摸。

在入园前，一方钱盒将配于手中，其中装有7枚**随盒赠钱**。

- 起用分队不同，赠钱装入类型与数量，或是钱盒容量亦不同。

- 在常规模式中，除**分队通宝**外，各分队初始钱盒组成相同。

  初始钱盒内钱币为：分队通宝，衡-奇土生金，花-火灼土沃，衡-金寒水衍，衡-投木炎延，衡-水生木护，衡-大炎通

可在底栏“钱盒”中查看盒中通宝。

每进入新的一层，或是往返于<岁兽残识>与<界园>，将立即行一次**投钱**：3枚通宝将被投出，并发挥效果，这些效果将持续至本层结束。

- 投出通宝将显示于底栏，亦会在钱盒中高亮，点击可再次确认。
- 若是觉得不合心意，进入钱盒，花上票券，可再行一次投钱。
  ※需注意：首次免费，再投花券，每次一张；进入新层，花销重置。
  ※还需注意：部分通宝效果仅可发挥一次，投出后若尝试再投，其将变为其它通宝。
- 客人若是收集了三枚以上的“花钱”或“厉钱”，在投出全部是“花钱”或“厉钱”时会相应呼出“花钱如愿”或是“厉钱行险”。

通宝可在行途战斗中掉落，也可在商人手中购买。

不过此时需丢掉一枚原有的通宝，换新获得的通宝入盒。

- ※需注意：若是换掉了正在生效层级效果的通宝，对应的效果也会消失！

获得通宝时，其将被概率付与**品相**，从而给予各种增益。

- 在高难度下，品相的出土概率也会提升。

点亮【古今学识】的《留钱篇》后，主动投钱时，可选择**留钱**：花费源石锭，保留最多2枚已投出的通宝，让未保留的通宝参与重投。

- 留1枚只需花费1源石锭，留2枚则需花费5源石锭。
- 被留下的通宝，其结果卡片底部将有绿色“留钱”标志，卡片上也会写明为“已保留”而非“已投出”。
- ※需注意：留下的通宝不会参与下次投钱，此时**不能**触发多次“被投出”效果，也不会因为这次重投而触发变化。

此外，部分奇遇中也会无消耗进行投钱检定，此时的通宝虽不会生效，但投出的币制会决定事件的结果好坏，请仔细配比。

- ※需注意：对于已投出的可变化通宝，**其在检定中仍然会以当前的原有状态参与检定，不会因为在检定中被投出而变化**。

### 4.岁时

在<界园>中，每当走过一定的层数，即会面临岁时。

- 在<请君入园·0+>时，进入第3层级时，若当前未曾面临岁时，触发一个新的岁时；
- 在<请君入园·2+>时，**改为**进入第2、4、6、7层级且面临的岁时数量小于4时，触发一个新的岁时。

岁时效果**永久生效**，并会跟随游人，在<岁兽残识>中继续生效。

随着难度等级提升，岁时效果也将增强。

触发第二，三个岁时之时，屏幕顶端层级后的图标样式也将发生变化。

通过某些方法，岁时将可**醒觉**，由混沌落于人间，其效果也将从不利转为有利。

### 5.化境

作战场地中，常会出现一些有着金色气息的地块，这便是**化境**地块。

不仅一些小玩意会在**化境**之上出现，一些通宝和收藏品也会对**化境**上的我方单位产生各种影响。

善用化境，或可有助力。

### 6.雕伥

在战场上，偶尔会出现雕伥。

这伥不攻击我方单位，但会每隔一段时间吐出额外的敌人，并且会让经过其周围四格的敌人获得一段时间内较高的移动速度和攻击速度加成。

而拆毁它也并非易事——其与其周围的敌人受到其周围四格范围外的我方单位的物理或法术伤害时，伤害量会大幅降低，而它也有着极高的耐久值。

不过，如果能够花上一笔部署费用，则可让它闭嘴两分半钟，这样它就不会再吐出敌人了。

如果能成功拆毁它，4源石锭的“伥怪遗宝”会作为本次作战的掉落物出现在结算结果中。

### 7.宝箱（葫芦）

不过您可能注意到了，在作战节点的战场上会随机出现各种“葫芦”，若是能**击破**它们，也可拿点小钱。

葫芦有三种，性情也不同：

- **宝葫芦**：普通的葫芦，击破后将额外带来2源石锭的奖励。
- **铁葫芦**：受到攻击时会反弹部分伤害。击破后将额外带来5源石锭的奖励。
- **怪葫芦**：通常拟态成[宝葫芦模样](https://prts.wiki/w/怪葫芦(装置))，受到攻击时将会[移动](https://prts.wiki/w/怪葫芦)。
  击破后将额外带来10源石锭的奖励，但若是让其进入目标点则会**偷走**10源石锭。
  而且这个葫芦会根据您持有的源石锭数量提升自身的**生命值**和**攻击力**。

### 8.票券

票券是一种较难获取的珍贵资源，可用于：

- 移动至侧方通路（消耗3票券）；
- 重新投钱决定本层通宝效果（首次免费，往后需花费1票券，新层重置）；
- 点亮【古今学识】的《绸缎篇》后，在<界园>的【诡意行商】中刷新商品（逐次上升）；
- 点亮【古今学识】的《纺锤篇》后，在<岁兽残识>的易与中刷新商品（逐次上升）。

票券的主要来源如下：

- 在<界园>的作战通过时，概率掉落票券；
- <岁兽残识>中，每个<祸乱>作战通过时，掉落1～2张票券；
- <岁兽残识>的<易与>中也会概率出售票券，售价2目标生命。

### 9.炎国阵营

您可能在<岁兽残识>中遭遇需要**炎国阵营干员**通过判定的情形。

- 在此环境下，炎国阵营干员指代所属阵营为**炎**或**炎-龙门**及其所属团队的干员。

## 古今学识

- 可在此处消耗[失落智识](https://prts.wiki/w/失落智识)为探索提供永久增益。

  除“游人背囊”类节点仅限“常规行动”模式生效外，所有加成在本主题的任何模式均生效，探索途中无法升级节点。

- 不同的节点能够带来不同方向上的加成：
  - 有的节点能拓展探索功能、带来多样化的探索经历；
  - 有的节点能增强属性或提升转换迷路幼伥的效率；
  - 有的节点能解锁特殊的分队流派，让探索更加多样。
  - 此外还有三个特殊的、不需要消耗失落智识升级的“游人背囊”类节点，其单个节点即拥有多种效果。
    当玩家点亮其任意一个前置节点后，**挑战对应难度时**即自动生效。
- 当所有节点都已点亮后，主界面入口处及该界面右上角将变为“全部解明”，且不再显示失落智识数量。同时在游戏结算页面相关信息会显示“MAX”，不再继续获得[失落智识](https://prts.wiki/w/失落智识)。
